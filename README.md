How to Build

1. Download zip을 한다

2. cocos2d-x 서브모듈 프로젝트로 가서 이것도 Download zip을 한다.

3. cocos2d-x 서브모듈 프로젝트 download한 압축 파일을 Robotopia/cocos2d 폴더에 압축해제한다.

4. 빌드한다.

5. 잘 된다.(아마)
